# AP Lang FRQ Reminder

A single-page HTML reminder for AP Language students: **Why you might be stuck on 4/6?**

## How to use

1. Upload `index.html` to a GitHub repository.
2. Turn on GitHub Pages in the repository settings.
3. Share the GitHub Pages link with students.

## Contents

- Clear thesis reminder
- Paragraph structure: Claim → Evidence → Commentary
- Evidence explanation prompts
- How to link commentary back to the thesis
- Sophistication strategies
- Final pre-submission checklist

## Files

- `index.html` — the complete webpage with all styling and illustrations included inline.
